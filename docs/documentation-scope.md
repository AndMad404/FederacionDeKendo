# Documentation scope

This repository is public and belongs to the Federation website product.

## Keep in this repository

- Product architecture and data-flow documentation.
- Accepted technical ADRs that affect this application.
- Development, verification, accessibility, testing, and release rules.
- Operator documentation such as calendar maintenance.
- Public content, SEO, deployment, and asset guidance.
- Technical review findings limited to this repository.

## Keep outside this repository

- Personal learning plans and self-assessments.
- AI tool comparisons, conversation history, and personal methodology.
- Portfolio planning and private case-study drafts.
- Notes about other repositories, local absolute paths, or private tooling.
- Secrets, private calendar URLs, tokens, credentials, and private attachments.

The private `DesarrolloAsistidoIA` repository is the source for the personal program and the Federation case-study working notes. Do not add links that require private access to understand or maintain this public project.

When a personal conclusion becomes an accepted technical decision for this application, document only the project-facing ADR here.
